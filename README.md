# Список отзывов IT компаний Казахстана
######  от разработчиков для разработчиков

## [Prime Source](https://p-s.kz/)
  
### Минусы:
  
  * Типичная галера с густо-населёнными опен-спейсом 
  * Проекты в основном финтех, много легаси, много говнокода, тяжело вносить изменения в проект, в следствии слабый контроль продукты и просроченые дедлайны
  * Отсутствует развитие и мотивация сотрудников, пересмотры зарплат, повышение возможно только для близких друзей руководства, избирательно.
  * Большая текучка кадров

### Плюсы:
  
  * Официальное трудоустройство
  * Чай-Кофе, завтраки на которые еще нужно успеть
  * Офис в благоприятной части города
